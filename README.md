# ☀️ SunCart – Summer Essentials Store

SunCart is a modern summer-themed eCommerce web application where users can explore and purchase seasonal products like sunglasses, skincare items, beach accessories, summer outfits, and more.

The platform provides a smooth and responsive shopping experience with authentication, protected routes, animated UI sections, and product details pages.

---

# 🌐 Live Website

🔗 Live URL: https://ph-b13-a08-suncart-summer-essential.vercel.app

---

# 🎯 Project Purpose

The goal of this project is to build a modern responsive summer eCommerce application using Next.js App Router and Better Auth while implementing authentication, protected routes, animations, dynamic routing, and reusable UI components.

---

# ✨ Key Features

- 🔐 Authentication using Better Auth
- 🔑 Google Sign In
- 👤 User Profile Page
- ✏️ Update User Information
- 🛍️ Summer Product Showcase
- 📄 Product Details Page
- 🔒 Protected Product Details Route
- 📱 Fully Responsive Design
- ⚡ Next.js App Router
- 🎨 Modern UI with Tailwind CSS + HeroUI
- ✨ React Spring Animations
- 🔔 Toast Notifications with React Toastify
- 🌅 Summer Themed Design
- 🚀 Deployed on Vercel
